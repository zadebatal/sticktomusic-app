# StickToMusic App

## Quick Start

1. Open terminal and navigate to this folder:
   ```
   cd sticktomusic-app
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. The app will open in your browser at http://localhost:3000

## Features

- Artist intake form
- Artist dashboard with metrics
- Operator dashboard with:
  - Artists management
  - World pages overview
  - Content scheduling with batch support
  - Content banks (hashtags/captions per aesthetic)
  - Application management

## Accessing Different Views

- **Home**: Landing page with pricing
- **Dashboard**: Click "Login" - artist view
- **Operator**: Click the small dot (•) next to Login
